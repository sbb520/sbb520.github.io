在ubuntu翻墙

2018-02-13 / sbb

一. 安装ssr客户端

wget http://www.djangoz.com/ssr
sudo mv ssr /usr/local/bin
sudo chmod 766 /usr/local/bin/ssr
ssr install
ssr config

具体的操作方式其实可以打开第一步下载的脚本看看的，开头有注释

二. 全局代理（最简单方法）

直接打开ubuntu的网络设置去进行设置。
setting->network->network proxy

三. chrome翻墙

下载SwitchySharp，添加到chrome扩展中，然后进行设置。
首先设置 Proxy Profiles设置里面的manual config里的SOCKS Host，根据自己的ssr去进行设置。

如果自己不需要浏览器全局代理，那么设置switch rules，根据自己的需要代理的网站去进行设置，其他不需要翻墙的网站则不需要代理，例如说我经常用谷歌和百度，就仅仅设置谷歌

使用的时候我一般使用Auto Switch Mode

如果遇到一些不常用的外国网站可以直接设置回自己浏览器全局代理模式。

四. 终端翻墙

使用privoxy进行代理，在配置文件（/etc/privoxy/config）进行如下设置

forward-socks5 / 127.0.0.1:1080 .
# local network do not use proxy
forward         192.168.*.*/    .
forward         10.*.*.*/       .
forward         127.*.*.*/      .

根本配置文件找到代理端口，搜索listen-address 看到端口是8118
然后重启服务，在终端输入以下命令即可

export http_proxy="http://127.0.0.1:8181"
export https_proxy=$http_proxy
export ftp_proxy=$http_proxy
export rsync_proxy=$http_proxy
export no_proxy="localhost,127.0.0.1"
